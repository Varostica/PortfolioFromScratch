# PortfolioFromScratch
New Portfolio built from scratch. Front-end technical assessment built with React, TypeScript, and Tailwind CSS, showcasing responsive UI, reusable components, and clean code practices.


## AI Usage

AI was used as a development assistant during this assessment to support planning, code organization, implementation review, and documentation. It helped with tasks such as refining component structure, suggesting improvements to service organization, identifying potential testing targets, and drafting technical explanations.

All AI-generated suggestions were reviewed, adapted, and validated before being included in the final solution. Final architectural decisions, integration choices, naming, and implementation details were selected and adjusted manually to fit the project requirements.
